# 💧 Neer Protection Toolkit

**Neer Protection Toolkit** is a powerful, lightweight CLI suite designed for Termux and Linux. It contains two ultimate tools:
1. **Neer Extractor:** Extract the front-end source code (HTML) of any website using its URL.
2. **Neer Protect:** Obfuscate and protect your Client-Side code (HTML, CSS, JS) using Advanced Compression and Base64 Encoding.

## ✨ Features
- 🌐 **Web Scraper (Extractor):** Download source code of any webpage
- 🚀 **Fast & Secure (Protect):** Encodes your code instantly
- 📱 **Mobile Friendly:** Perfect for Termux and narrow screens
- 🔄 **Auto-Decoding:** Browser dynamically decodes and renders
- 🎨 **Beautiful UI:** Python's `rich` library for elegant CLI
- 🔔 **YouTube:** Subscribe: https://youtube.com/@hackerneer

## 🛠️ Prerequisites
```bash
pkg update && upgrade
pkg install python
pkg install git
git clone https://github.com/Hackerneer/Neer-Protection-Toolkit.git
cd Neer-Protection-Toolkit
pip install -r requirements.txt
```


Select the HTML file you want to protect and it will generate neerprotected_[filename].html

📁 File Naming Convention

· Extractor Output: neer_example_com_source.html
· Protector Output: neerprotected_index.html

🛑 Disclaimer

This tool is intended for basic code protection and educational purposes. Advanced reverse engineers can still decode client-side code. Do not rely on this for highly sensitive backend credentials.
